영지 목록

모바일에서는 맵을 숨기고 통계/목록만 표시합니다. PC나 태블릿 가로 화면에서 맵을 확인해 주세요.

Y 위치 보정

좌상단 기준으로 전체 영지 좌표의 Y값을 한 번에 이동합니다. 음수면 위로, 양수면 아래로 이동합니다.

서버 구성

대표 서버 1개와 대진 서버 4개를 입력해 주세요. 다섯 칸 모두 #1234 형식으로 입력해 주세요. 서버 리스트에는 대표 서버만 표시됩니다.

서버 관리자 설정

현재 선택된 대표 서버 기준으로 서버관리자 계정을 추가합니다. 이메일/비밀번호로 Auth 계정을 만들고, 서버 권한도 함께 등록합니다.

영지 점령 변경

영지를 선택하여 점령 연맹을 변경해 주세요.

기존 연맹 진영 일괄 지정

슈퍼관리자만 사용할 수 있습니다. 기존 등록된 연맹의 진영을 한 번에 질서/복수로 지정하여 저장합니다.

서버 관리자 추가 요청

신청 정보를 입력해 주세요. 슈퍼관리자가 검토 후 계정을 생성해 드립니다.

서버 관리자 신청 목록

서버 관리자 신청 결과

❌ 신청이 거절되었습니다.

다시 신청하시려면 [서버 관리자 신청] 버튼을 이용해주세요.

✅ 서버 관리자 승인 완료!

관리자 권한이 승인되었습니다. 아래 순서에 따라 로그인하고 시작해 주세요.

1화면 우측 상단 [관리자 모드] 버튼을 클릭합니다.
2모드 선택창에서 2 를 입력하고 확인을 누릅니다. (서버 관리자 로그인)
3신청한 이메일을 입력합니다.
4신청한 비밀번호를 입력합니다.
5로그인 후 좌측 메뉴 [연맹 추가] 버튼으로 연맹을 등록합니다.
6지도 위 영지를 클릭하면 점령 연맹과 세력 정보를 변경할 수 있습니다.

사용법

🗺️ 지도 영지를 클릭하면 해당 영지의 정보(유형, 점령 연맹, 세력치 등)가 좌측 패널에 표시됩니다.

🔍 레벨 필터 버튼을 눌러 특정 레벨 영지만 지도에 표시할 수 있습니다. 전체를 선택하면 모든 영지가 표시됩니다.

🏷️ 연맹 필터 전체 / 미점령 / 점령 또는 특정 연맹 버튼을 눌러 해당 조건의 영지만 볼 수 있습니다.

📊 현황 통계 질서/복수 세력치 합계와 연맹별 순위를 확인할 수 있습니다. 상단 바는 질서(파랑) vs 복수(빨강) 세력 비율을 시각적으로 보여줍니다.

➕ 연맹 추가 좌측 상단의 [연맹 추가] 버튼을 클릭하면 연맹 이름과 진영(질서/복수)을 입력할 수 있습니다. 등록된 연맹은 연맹 필터와 순위에 자동으로 반영됩니다. ⚠️ 연맹 추가는 해당 서버의 서버 관리자 또는 슈퍼관리자만 가능합니다.

🛡️ 서버 관리자 신청 전황판을 직접 관리하고 싶으시면 [서버 관리자 신청] 버튼을 통해 신청할 수 있습니다. 로그인하지 않은 상태에서만 버튼이 표시됩니다. 신청 시 다음 정보를 입력해 주세요. • 본인 서버 번호 (#1234 형식) • 닉네임 • 사용할 이메일 주소 • 비밀번호 (6자 이상) • 관리할 서버 5개 번호 (추가할 서버, 질서 서버1/2, 복수 서버1/2) 신청 후 슈퍼관리자가 검토하여 계정을 생성해 드립니다. 승인되면 [관리자 모드] 버튼으로 로그인하여 전황판을 관리할 수 있습니다.
