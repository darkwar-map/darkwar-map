현황 통계

영지 목록

모바일에서는 맵을 숨기고 통계/목록만 표시해. PC나 태블릿 가로 화면에서 맵을 확인해줘.

Y 위치 보정

좌상단 기준으로 전체 영지 좌표의 Y값을 한 번에 이동해. 음수면 위로, 양수면 아래로 이동해.

서버 구성

대표 서버 1개와 대진 서버 4개를 입력해. 다섯 칸 모두 #1234 형식으로 입력해. 서버 리스트에는 대표 서버만 표시돼.

서버 관리자 설정

현재 선택된 대표 서버 기준으로 서버관리자 계정을 추가해. 이메일/비밀번호로 Auth 계정을 만들고, 서버 권한도 같이 등록해.

영지 점령 변경

영지를 선택해 점령 연맹을 바꿔줘.

기존 연맹 진영 일괄 지정

슈퍼관리자만 사용할 수 있어. 기존 등록된 연맹의 진영을 한 번에 질서/복수로 지정해 저장해.

사용법

🗺️ 지도 영지를 클릭하면 해당 영지의 정보(유형, 점령 연맹, 세력치 등)가 좌측 패널에 표시됩니다.

🔍 레벨 필터 버튼을 눌러 특정 레벨 영지만 지도에 표시할 수 있습니다. 전체를 선택하면 모든 영지가 표시됩니다.

🏷️ 연맹 필터 전체 / 미점령 / 점령 또는 특정 연맹 버튼을 눌러 해당 조건의 영지만 볼 수 있습니다.

📊 현황 통계 질서/복수 세력치 합계와 연맹별 순위를 확인할 수 있습니다. 상단 바는 질서(파랑) vs 복수(빨강) 세력 비율을 시각적으로 보여줍니다.

➕ 연맹 추가 연맹 이름과 진영(질서/복수)을 입력하면 필터와 순위에 반영됩니다.
